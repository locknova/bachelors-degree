# Дипломная работа / bachelors-degree

В данном проекте я буду сохранять данные по мере выполнения моей дипломной работы (ВКР).
Есть 4 осовных раздела:

### Задачи и тема работы (tasks_theme.txt)
В файле tasks_theme.txt я укажу цели, задачи, рамки работы и сформулизую тему

### Структура работы (structure.txt)
В файле structure.txt я укажу ту структуру работы, которую требуется предоставить для её сдачи в моём вузе

### Список литературы, источников (literature.txt)
В файле literature.txt я укажу источники и литературу для работы. Мой анализ их и краткая сводка по содержанию

### Работа, ВКР (final_bachelors_work.docx)
В файле final_bachelors_work.docx я буду писать ВКР, используя вышеописанные наработки

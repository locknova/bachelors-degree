# Дипломная работа / bachelors-degree

В данном проекте я буду сохранять данные по мере выполнения моей дипломной работы (ВКР).

### Задачи и тема работы (tasks_theme)
В файле tasks_theme я укажу цели, задачи, рамки работы и сформулизую тему

### Структура работы (structure)
В файле structure я укажу ту структуру работы, которую требуется предоставить для её сдачи в моём вузе

### Список литературы, источников (literature)
В файле literature я укажу источники и литературу для работы. Мой анализ их и краткая сводка по содержанию

### Работа, ВКР (final_bachelors_work)
В файле final_bachelors_work я буду писать ВКР, используя вышеописанные наработки
